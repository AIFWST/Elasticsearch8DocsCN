## 搜索接口

搜索 API 用于搜索和聚合存储在 ElasticsearchIndex 和数据流中的数据。有关概述和相关教程，请参阅搜索数据。

大多数搜索 API 都支持多目标语法，但解释 API 除外。

### 合作研究

* 搜索 * 多重搜索 * 异步搜索 * 时间点 * 建议者 * 术语枚举 * 滚动 * 清除滚动

### 搜索测试

* 解释 * 现场能力 * 简介 * 排名评估 * 搜索分片 * 验证

### 搜索模板

* 搜索模板 * 多搜索模板 * 渲染搜索模板
