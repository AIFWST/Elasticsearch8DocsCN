## 显示目录

**Synopsis:**

    
    
    SHOW CATALOGS

**描述**：列出可用目录及其类型。

    
    
    SHOW CATALOGS;
    
         name         |     type
    ------------------+---------------
    javaRestTest         |local
    my_remote_cluster |remote
