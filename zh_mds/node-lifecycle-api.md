## 节点生命周期接口

此功能专为 ElasticsearchService、Elastic Cloud Enterprise 和 Kubernetes 上的 ElasticCloud 间接使用而设计。不支持直接使用。

您可以使用关闭 API 为临时或永久关闭的节点做好准备，监控关闭状态，并启用以前关闭的节点以恢复正常操作。

### 节点关机接口

* 放置关机接口 * 获取关机接口 * 删除关机接口
