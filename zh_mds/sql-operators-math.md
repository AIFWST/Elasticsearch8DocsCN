## 数学运算符

执行影响一个或两个值的数学运算。结果是数值类型的值。

### '添加(+)'

    
    
    SELECT 1 + 1 AS x;

### '减去(中缀-)'

    
    
    SELECT 1 - 1 AS x;

### '否定(一元-)'

    
    
    SELECT - 1 AS x;

### '乘法(*)'

    
    
    SELECT 2 * 3 AS x;

### '除法(/)'

    
    
    SELECT 6 / 3 AS x;

### 'Modulo orRemainder(%)'

    
    
    SELECT 5 % 2 AS x;
