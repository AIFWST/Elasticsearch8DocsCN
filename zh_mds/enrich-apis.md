## 扩充接口

以下扩充 API 可用于管理扩充策略：

* 创建扩充策略以创建或替换扩充策略 * 删除扩充策略以删除扩充策略 * 获取扩充策略以返回有关扩充策略的信息 * 执行扩充策略以执行扩充策略 * 丰富统计信息以获取与丰富相关的统计信息
