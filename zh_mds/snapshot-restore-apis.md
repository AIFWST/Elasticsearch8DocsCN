## 快照和恢复接口

您可以使用以下 API 设置快照存储库、管理快照备份以及将快照还原到正在运行的集群。

更多信息，请参见快照与还原。

### 快照仓库管理接口

* 创建或更新快照存储库 * 验证快照存储库 * 获取快照存储库 * 删除快照存储库 * 清理快照存储库

### 快照管理接口

* 创建快照 * 克隆快照 * 获取快照 * 获取快照状态 * 恢复快照 * 删除快照
